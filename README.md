# 🐍 Curso de Python - Cilsa

Este repositorio contiene mis avances y ejercicios resueltos del curso de **Introducción al desarrollo de software con Python** en la plataforma de Cilsa. El objetivo es mantener una referencia completa y accesible del contenido del curso, especialmente en caso de no tener acceso al campus en el futuro.

## 📚 Contenido del Curso

El curso abarca temas fundamentales de Python, avanzando desde los conceptos básicos hasta temas más complejos como POO y bases de datos. A continuación, se detalla la organización semanal de los contenidos:

- **Semana 1**: Variables y Tipos de Datos
- **Semana 2**: Estructuras de Control Condicionales
- **Semana 3**: Estructuras de Control Repetitivas
- **Semana 4**: Tipos de Datos Complejos
- **Semana 5**: Manejo de Excepciones
- **Semana 6**: Diccionarios
- **Semana 7**: Funciones
- **Semana 8**: Archivos
- **Semana 9**: Procesamiento de Datos
- **Semana 10**: Módulos y Librerías
- **Semana 11**: Programación Orientada a Objetos (POO)
- **Semana 12**: 📝 Proyecto Final
- **Semana 13**: Bases de Datos
- **Semana 14**: 🎉 Cierre de Curso

Cada semana incluye:
- **Desafíos resueltos**: 🏅 Ejercicios prácticos para reforzar los conocimientos de cada tema.
- **Material complementario**: 📄 Documentación y recursos adicionales proporcionados en el campus.

## 🎯 Objetivos del Curso

El curso se enfoca en:
- ✅ Comprender la sintaxis y el uso de Python.
- ✅ Desarrollar habilidades en la programación orientada a objetos.
- ✅ Aplicar estructuras de control y funciones en la resolución de problemas.
- ✅ Manipular datos en archivos y bases de datos para integrarse con otros sistemas.

## 🗂 Organización del Repositorio

El repositorio está organizado por carpetas semanales, donde cada una corresponde a una clase y tema específico del curso. Dentro de cada carpeta encontrarás:
- Los **desafíos resueltos** correspondientes.
- Material adicional para profundizar en cada tema.

## 📌 Notas Finales

Este repositorio es una herramienta para consolidar lo aprendido y servirá de referencia a largo plazo.
